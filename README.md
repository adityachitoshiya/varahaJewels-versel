# Varaha Jewels

**Where Heritage Meets Royalty**

A premium e-commerce website for Varaha Jewels - showcasing luxury heritage-inspired jewelry crafted with tradition and artistry.

## Features

- 🎨 **Premium Design** - Elegant UI with warm sand color palette
- 🛒 **Full E-commerce** - Product catalog, cart, and checkout
- 💳 **Payment Integration** - Razorpay with COD support
- 📱 **Responsive** - Mobile-first design
- ⚡ **Fast Performance** - Built with Next.js 14
- 🎯 **Launch Countdown** - 39-day countdown timer
- 🔐 **Admin Panel** - Order and product management
- ✅ **Testimonials** - Verified customer reviews
- 🎁 **Coupon System** - TESTADI test coupon

## Tech Stack

- **Framework:** Next.js 14.2.0
- **UI:** React 19.2.0, Tailwind CSS
- **Icons:** Lucide React
- **Payments:** Razorpay
- **Storage:** Local JSON files
- **Deployment:** Vercel ready

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Test Credentials

- **Coupon Code:** TESTADI (Makes order ₹1)
